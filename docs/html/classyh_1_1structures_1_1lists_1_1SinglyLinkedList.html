Detailed Description

template<typename T>
class yh::structures::lists::SinglyLinkedList< T >

A list implemented by singly-linked nodes.

Parameters
TThe data type to be held by the list.
Warning
This class borrows elements but do not own them: The user should be responsible for deleting them.
