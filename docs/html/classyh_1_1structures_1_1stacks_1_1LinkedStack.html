Detailed Description

template<typename T>
class yh::structures::stacks::LinkedStack< T >

A stack implemented by singly-linked list.

Parameters
TThe data type to be held by the stack.
Warning
This class borrows elements but do not own them: The user should be responsible for deleting them.
