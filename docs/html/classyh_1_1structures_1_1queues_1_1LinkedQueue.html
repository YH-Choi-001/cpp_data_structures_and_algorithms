Detailed Description

template<typename T>
class yh::structures::queues::LinkedQueue< T >

A queue implemented by singly-linked list.

Parameters
TThe data type to be held by the queue.
Warning
This class borrows elements but do not own them: The user should be responsible for deleting them.
